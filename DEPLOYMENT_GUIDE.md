GitHub Pages upload steps:
1. Create a new repository on GitHub.
2. Upload this index.html file.
3. If you also want to keep the backup assets folder, upload it too.
4. Go to Settings -> Pages.
5. Set Source to Deploy from a branch.
6. Choose branch: main, folder: / (root).
7. Save and wait for the live URL.
8. Add that URL to resume, LinkedIn Featured, and Contact info.
